## 8. 데이터 접근 및 재현성

### 8.1 코드 저장소
- **GitHub**: https://github.com/yonghwan1106/compound_climate_project
- 데이터 수집 스크립트: `src/data_collection/`
- 전처리 코드: `src/preprocessing/`
- CMIP6 분석 코드: `src/analysis/future_scenario.py`

### 8.2 환경 설정
```bash
# 가상환경 생성
conda create -n climate python=3.10
conda activate climate

# 패키지 설치
pip install -r requirements.txt

# 데이터 수집 실행
python -m src.data_collection.kma_collector
python -m src.data_collection.era5_collector
python -m src.data_collection.socioeconomic_collector

# CMIP6 미래 시나리오 분석
python src/analysis/future_scenario.py
```

### 8.3 API 설정
```bash
# 기상청 API 키 설정
export KMA_API_KEY="your_api_key"

# Copernicus CDS API 설정 (~/.cdsapirc)
url: https://cds.climate.copernicus.eu/api
key: your_uid:your_api_key
```

### 8.4 데이터 라이선스 준수
- 모든 데이터는 각 기관의 이용약관에 따라 활용
- 공공누리 제1유형: 출처 표시 후 자유 이용
- Copernicus License: 비상업적 연구 목적 이용 가능
- 상업적 이용 시 별도 허가 필요 여부 확인 필수
