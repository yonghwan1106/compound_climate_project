9 ▪ 시뮬레이션 기상 데이터
  • 모델 학습/검증용 합성 기상 시계열
  • 생성 방법: 실제 관측 데이터 기반 Monte Carlo 시뮬레이션
  • 데이터 규모: 10,000 시계열 × 30일
  • 변수: 기온, 강수량, 습도, 기압, 풍속
  • 활용 목적: 모델 학습 데이터 증강
 AI 합성데이터 ▪ AI 합성 방법:

1. 실제 관측 데이터의 통계적 분포 추정
2. Copula 함수로 변수 간 의존성 모델링
3. Monte Carlo 샘플링으로 합성 시계열 생성
4. 극한 이벤트 발생 확률 조정

▪ 파일명: data/synthetic/simulated_weather.csv
10 ▪ 취약성 평가 학습용 합성 데이터
  • 영향 예측 모델 학습용 합성 데이터셋
  • 생성 방법: 실제 피해 데이터 기반 Bootstrap 리샘플링 + 노이즈 추가
  • 데이터 규모: 5,000 레코드
  • 변수: 기후변수, 사회경제변수, 피해규모
  • 활용 목적: 영향 예측 모델 과적합 방지
 AI 합성데이터 ▪ AI 합성 방법:

1. 실제 피해 데이터에서 Bootstrap 샘플링
2. Gaussian 노이즈 추가 (σ=0.1)
3. SMOTE로 희귀 클래스 오버샘플링
4. 이상치 필터링 및 정규화

▪ 파일명: data/synthetic/vulnerability_train.csv
