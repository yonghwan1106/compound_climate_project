## 7. 데이터 접근 및 재현성

### 7.1 코드 저장소
- GitHub: `https://github.com/[anonymous]/compound-climate-korea`
- 데이터 수집 스크립트: `src/data_collection/`
- 전처리 코드: `src/preprocessing/`

### 7.2 환경 설정
```bash
# 가상환경 생성
conda create -n climate python=3.10
conda activate climate

# 패키지 설치
pip install -r requirements.txt

# 데이터 수집 실행
python -m src.data_collection.kma_collector
python -m src.data_collection.era5_collector
python -m src.data_collection.socioeconomic_collector
```

### 7.3 데이터 라이선스 준수
- 모든 데이터는 각 기관의 이용약관에 따라 활용
- 상업적 이용 시 별도 허가 필요 여부 확인 필수
- 출처 표기 의무 준수
